# Hello ImGui

This is a "Hello World" for ImGUI. ImGUI requires a backend.
SFML works as the backend.

This project contains the boilerplate code for imgui-SFML bindings
for an easier integration between both.

## Versions

| Tool  | Version |
|:------|:--------|
| CMake | 3.20    |
| std   | c++20   |
| conan | 1.62    |
| fmt   | 10.0.0  |
| ImGUI | 1.90.5  |
| SFML  | 2.6.1   |
